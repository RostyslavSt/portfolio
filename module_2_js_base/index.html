vanilla js

  1. Переопределить встроенную функцию alert: при вызове alert('message') на страницу добавляется стилизованный блок с текстом, через три секунды блок исчезает.

  2. Написать функцию, которая запускает окно prompt, требующее ввести число. Prompt всплывает до тех пор, пока пользователь не нажмет Отмена. Все значения, которые пользователь ввел, запоминаются в массив. Функция возвращает наименьшее из них.

  3. Напишите функцию, которая рассчитывает среднее арифметическое из всех переданных параметров, количество которых не ограничено.

  4. Сформируйте строку с шахматной доской из вложенных циклов. Для перевода строки используйте \n. Код должен поддерживать легкое изменение размеров доски.

  5. Создайте функцию-конструктор createUser, полями объекта являются firstname, lastname и fullname. Поля firstname и lastname запрашиваются в окне prompt, поле fullname является конкатенацией имени и фамилии. Результатом работы функции является новый объект со введенными полями. Новый объект представляется на странице в виде списка значений или таблицы. Если создавать пользователей в цикле, то на странице выводится полный список.

  6. В функциональном стиле (new) напишите класс Calculator. Входящий параметр - флаг true/false, обозначающий, включен калькулятор или выключен. По умолчанию (если параметр не передан) калькулятор включен. Определить метод для включения-выключения калькулятора. Определить функции sum, diff, div, multi, mod (остаток от деления), pow (возведение степень). Если калькулятор выключен, то расчет не происходит, если включен – возвращается результат вычисления.   7. Создайте двумерный массив, в котором хранятся позиции закрашенных квадратов. В цикле переберите массив и изобразите картинку в html
